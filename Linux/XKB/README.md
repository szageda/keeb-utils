# 1 XKB

I've created very barebones XKB configuration files. They only contain the Colemak-DH base layout, but no additional layers.

## 1.1 Installation

Follow these steps to **install the Colemak-DH base layout XKB configuration files:**

1. Navigate to `keeb-utils\Linux\XKB` in your command line interface.

2. Run this command to install the XKB configuration files: `sudo make install`.

3. Restart your display manager or your computer.

4. Go to your keyboard settings and add `English (Colemak-DH Keeb Utils)`.

**Notes:**  
Run these commands from the command line in `keeb-utils\Linux\XKB` to:  
\- **remove:** `sudo make uninstall`  
\- **reinstall:** `sudo make reinstall`
